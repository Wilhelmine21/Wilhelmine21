# Light Number
* 那什麼是Light Number？簡單來說，它就是一個算術權重小的數字。Light Number是由2的次方數來進行加減來得出的。
* Example:

	</br><img src="./img/Pasted image 20220414110354.png" width="50%" height="50%"/></br>
	
	|s1 |m1 |s2 |m2 |s3 |m3|
	|:----:|:----:|:----:|:----:|:----:|:----:|
	|+ |0 |+ |-1 |+ |-3|
	|+ |-1 |- |-3 |+ |-4|
